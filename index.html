Ph.D. Student in Computer Science

I am a Combined Master and Ph.D. student at UNIST (Ulsan National Institute of Science and Technology) in the CSSL lab, advised by Prof. Hyungon Moon. My research focuses on building secure and efficient system architectures.

View Publications Contact Me

Research Interests

System Security

Mitigating memory corruption vulnerabilities (e.g., Use-After-Free) at the OS level.

Operating Systems

Designing efficient memory allocators, sweeping architectures, and temporal safety mechanisms.

Memory Management

Optimizing caching and resource isolation for serverless and in-storage computing environments.

Program Analysis

Applying Static Value-Flow Analysis (SVF) and LLVM to detect software flaws systematically.
